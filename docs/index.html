How to subscribe:
  1. Copy the calendar link above
  2. On iPhone/iPad: Settings > Calendar > Accounts > Add Account > Other > Add Subscribed Calendar
  3. On Mac: File > New Calendar Subscription
  4. Paste the link and click Subscribe
