How to subscribe using your own calendar app:

  1. Click "Copy Calendar Link" below
  2. iPhone/iPad: Settings > Apps > Calendar > Accounts > Add Account > Other > Add Subscribed Calendar
  3. Mac: Open Calendar app > File > New Calendar Subscription
  4. Paste the link and tap/click Subscribe

  2. Go to calendar.google.com (desktop only)
  3. Click + next to "Other calendars" in the left sidebar
  4. Select "From URL" and paste the link

Note: Google Calendar subscriptions must be added from a computer, not the mobile app.

  1. Click "Copy Calendar Link" below
  2. Outlook.com: Settings (gear) > View all Outlook settings > Calendar > Shared calendars > Subscribe from web
  3. Outlook desktop: Add Calendar > From Internet, then paste the link
  4. Name the calendar and click Import/Subscribe
